Cara Menggunakan Simulasi
Tekan tombol "Mulai" / "Start" untuk menjalankan animasi gelombang.

Pilih Nada Harmonik
Pilih mode:
Harmonik 1 (n = 1)
Harmonik 2 (n = 2)
Harmonik 3 (n = 3)
Harmonik 4 (n = 4)
Perhatikan pola gelombang yang terbentuk.

Amati Pola Gelombang
Perut (P) → bagian gelombang dengan simpangan maksimum
Simpul (S) → bagian gelombang dengan simpangan nol

👉 Pada pipa organa terbuka:
Kedua ujung selalu Perut (P)

Untuk nada harmonik tetap, geser slider untuk mengubah panjang pipa
amati frekuensi nada harmoniknya tersebut!
